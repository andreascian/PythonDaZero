Per vedere i percorsi di ricerca dei moduli di python si fa:

>>> import sys
>>> dir(sys)
>>> for i in sys.path:
...     print(i)
... 

/usr/lib/python35.zip
/usr/lib/python3.5
/usr/lib/python3.5/plat-x86_64-linux-gnu
/usr/lib/python3.5/lib-dynload
/usr/local/lib/python3.5/dist-packages
/usr/lib/python3/dist-packages
>>> 

il modulo os contiene i comandi di interfaccia con il sistema operativo

>>> import os
>>> dir(os)

tra python 2 e python 3 ci sono alcune differenza nella gestione del packages.
nel file __init__.py in python 3 si deve importare esplicitamente il path
corrente.
